# Challenge Theme: Connecting the Dots Through Docs

Extracting a structured outline of any document — essentially the Title, and headings like H1, H2, and
H3 — in a clean, hierarchical format.

### Constraint Requirement
| Criteria | Constraint |
| -------- | ---------- |
| Execution time | ≤ 10 seconds for a 50-page PDF |
| Model size | ≤ 200MB (if used) |
| Network | No internet access allowed |
| Runtime | Must run on CPU (amd64), solution should run on the system with 8 CPUs and 16 GB RAM configurations |
